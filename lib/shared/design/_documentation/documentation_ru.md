# Дизайн-система

Этот каталог содержит дизайн-систему приложения.

## Назначение
- Визуальная консистентность
- Централизованные стили
- Поддержка светлой и тёмной темы

## Цвета
- Запрещено использовать Color напрямую
- Только цвета из дизайн-системы
- Цвет добавляется сразу в обе темы

## Типографика
- Основана на Open Sans
- Использовать только Theme.textTheme
- Не создавать локальные TextStyle
