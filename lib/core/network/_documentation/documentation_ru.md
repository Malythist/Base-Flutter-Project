# Network слой

Этот каталог содержит сетевую конфигурацию проекта.

## Назначение

Network слой:
- Централизует настройки API
- Хранит базовый URL
- Изолирует сеть от фич

## ApiConfig

ApiConfig определяет:
- Базовый URL API
- Конфигурацию окружений

ApiConfig всегда должен приходить через DI.
Создавать его вручную запрещено.
