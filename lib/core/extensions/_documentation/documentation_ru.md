# Core extensions

Этот каталог содержит переиспользуемые расширения Dart и Flutter.

## Философия

Экстеншены должны:
- Быть маленькими и понятными
- Уменьшать количество кода
- Не содержать бизнес-логики
- Быть безопасными для использования везде

Если логика усложняется — это уже не экстеншен.

## Theme extensions

Упрощённый доступ к теме.

Пример:
context.colors.primary
context.text.titleMedium

Назначение:
- Убрать Theme.of(context)
- Повысить читаемость UI

## AsyncValue extensions

Упрощение рендера AsyncValue.

Пример:
async.whenWidget(...)

Назначение:
- Убрать дублирование when(...)
- Сделать экраны декларативными

## Image extensions

Обёртка над cached_network_image.

Назначение:
- Централизовать загрузку изображений
- Использовать кэш и плейсхолдеры

## Snackbar extensions

Назначение:
- Единый способ показывать сообщения пользователю
